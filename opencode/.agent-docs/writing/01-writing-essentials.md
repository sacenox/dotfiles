# Writing Clearly and Concisely

## Overview

Write with clarity and force. Be specific and concrete.

## When to Use

Use this guidance whenever you write prose for humans:
- Documentation, README files, technical explanations
- Commit messages, pull request descriptions
- Error messages, UI copy, help text, comments
- Reports, summaries, or any explanation
- Editing to improve clarity

## Core Principles (Strunk)

- Use active voice.
- Put statements in positive form.
- Use definite, specific, concrete language.
- Omit needless words.
- Keep related words together.
- Place emphatic words at the end of the sentence.
- One paragraph per topic.
- Begin paragraphs with a topic sentence.
